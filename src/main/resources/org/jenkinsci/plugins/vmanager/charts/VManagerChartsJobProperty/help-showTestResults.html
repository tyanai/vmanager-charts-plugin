Stacked bar chart showing the regression-health anomaly summary per build: the number of failures classified as anomalies vs. those flagged as related to known issues, plus the unclassified remainder.

Prerequisites:

  • A valid Verisium AI license must be available on the vManager server.
  • Regression Health must be configured and enabled in vManager for the project / regressions whose sessions this job runs, so that anomaly classifications are produced for each session.

Without these in place the chart will be empty (or show only the unclassified bucket) because vManager has no anomaly data to return.
